    \begin{enumerate}
      \item Un message d'erreur est affiché
    \end{enumerate}\\ \hline

      \item Le station de base informe le robot de son non repérage.

   \textbf{Cas d'utilisation:} & Déterminer son objectif  \\ \hline
   \textbf{Système:}  &  Pirate des Caraïbes: Équipe 2 \\ \hline
   \textbf{Acteur(s):} & Robot, station de base et station de recharge \\ \hline
    \textbf{Parties prenantes et intérêts:} & Robot: Il doit connaître la lettre codée par la station de recharge afin de connaître l'île destination. \\ \hline
    \textbf{Préconditions:} &  Le robot est en mesure de communiquer avec la station de recharge. \\ \hline
    \textbf{Postconditions:} &  Le robot connaît la description de l'île de destination. \\ \hline
    \textbf{Scénario principal:} &  \begin{enumerate}
	\item Le robot trouve la station de recharge à l'aide de sa caméra.
    	\item Le robot détermine la trajectoire à prendre (calculer un chemin).
    	\item Le robot effectue le déplacement (déplacer le robot) jusqu'à la station de recharge.
    	\item La station de recharge envoie un code Manchester correspondant à la lettre code de description de l'île.
    	\item Le robot décode le code Manchester pour retrouver une lettre.
    	\begin{enumerate}
	    \item Le robot transforme les bits en nombre de 0 à 127.
	    \item Le robot trouve le caractère ASCII correspondant à ce nombre.
    	\end{enumerate}
    	\item Le robot envoie cette lettre à la station de base
    	\item La station de base affiche la lettre 
    	\item La station de base envoie une requête utilisant la lettre au serveur de description de forme.
    	\item La station de base envoie la réponse du serveur en JSON au robot
    	\item La station de base affiche la réponse correspondant à la description d'une île.
    \end{enumerate}  \\ \hline
    \textbf{Extensions:} &  Aucune. \\ \hline
    \textbf{Interrogations:} & Quel sera le mode de communication du robot et de la station de recharge?  \\ \hline

    	\item Le robot se déplace jusqu'à l'île (Déplacer le robot) en soulevant le trésor.
    	\item Le robot se positionne afin que le trésor puisse être déposer sur l'île
    	\item Le robot dépose le trésor sur l'ile.
